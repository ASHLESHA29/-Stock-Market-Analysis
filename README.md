#📊 Stock Market Analysis -

📌 Project Overview:-
This project focuses on analyzing stock market data using Python in Jupyter Notebook.
The objective of this project is to understand stock trends, price movements, volatility, moving averages, daily returns, and correlation between multiple stocks through data analysis and visualization techniques.

The project uses real-world stock market data fetched from Yahoo Finance and performs Exploratory Data Analysis (EDA) to generate meaningful insights.

##🎯 Objectives:-
Analyze historical stock price data
Visualize stock trends over time
Calculate moving averages (MA50 & MA200)
Analyze daily returns and volatility
Understand stock correlation between multiple companies
Generate insights using data visualization

##🛠️ Technologies & Libraries Used:-
Python
Jupyter Notebook
Pandas
NumPy
Matplotlib
Seaborn
yFinance

##📥 Data Collection:-
Stock market data was collected using the yfinance library from Yahoo Finance.

Example stocks analyzed:-
TCS
Infosys
Reliance

##📊 Project Workflow:-
1. Importing Libraries
Imported required Python libraries for data analysis and visualization.
2. Data Collection
Fetched historical stock data using Yahoo Finance API.
3. Data Cleaning & Understanding
Performed:
Missing value checks
Data inspection
Statistical summary analysis
4. Stock Price Visualization
Visualized stock closing prices to understand overall market trends.
5. Moving Average Analysis
Calculated:
50-Day Moving Average (MA50)
200-Day Moving Average (MA200)
Purpose:
Identify short-term and long-term market trends
Detect bullish and bearish signals
6. Daily Returns Analysis
Analyze stock volatility
Understand risk and return behavior
7. Correlation Analysis
Analyzed correlation between multiple stocks using a heatmap.
Purpose:
Understand relationships between stocks
Identify diversification opportunities
📈 Visualizations Included
Closing Price Trend
Moving Average Graph
Daily Returns Histogram
Correlation Heatmap


##🔍 Key Insights:-
IT sector stocks like TCS and Infosys showed strong positive correlation.
Moving averages helped identify market trends and crossover signals.
Daily return analysis showed that most stock returns were concentrated near 0%, indicating normal day-to-day fluctuations.
Correlation analysis demonstrated diversification opportunities between stocks from different sectors.
